# Booking System - Ամրագրման համակարգ

Պարզ ամրագրման համակարգ, որն աշխատում է քո համակարգչի վրա։

## 🚀 Ինչպես գործարկել (3 քայլ)

### Քայլ 1: Տեղադրիր Node.js-ը (միայն առաջին անգամ)

1. Գնա՝ **https://nodejs.org**
2. Սեղմիր մեծ **կանաչ LTS կոճակը**
3. Ներբեռնված `.msi` ֆայլը բացիր
4. Սեղմիր **Next → Next → Next → Install**
5. Վերագործարկիր համակարգիչը

### Քայլ 2: Ներբեռնիր և բացիր zip-ը

Ներբեռնիր `booking-app-windows.zip` ֆայլը։ Սեղմիր աջ մկնիկով → **"Extract All..."** → ընտրիր տեղ (օրինակ՝ Desktop)։

### Քայլ 3: Գործարկիր

Մտիր `booking-app-windows` թղթապանակի մեջ և **կրկնակի սեղմիր** `Start-BookingApp.bat` ֆայլի վրա։

- **Առաջին անգամ:** 1 րոպե կսպասես (dependencies տեղադրվում են)
- **Հետագա անգամները:** անմիջապես բացվում է

Բրաուզերը ինքը կբացվի հետևյալ էջով՝
- **Ամրագրման էջ:** http://localhost:5000
- **Admin panel:** http://localhost:5000/admin

## 🛑 Կանգնեցնելու համար

Պարզապես **փակիր սև պատուհանը** (terminal-ը)։

---

## ❓ Խնդիրներ ու լուծումներ

### "Node.js is not installed" սխալ
→ Տեղադրիր Node.js-ը (Քայլ 1) և վերագործարկիր համակարգիչը։

### "Windows protected your PC" զգուշացում
→ Սեղմիր **"More info"** → **"Run anyway"**։
Սա նորմալ է անստորագիր `.bat` ֆայլերի համար։ Ֆայլը ապահով է։

### "Port 5000 is already in use"
→ Մեկ այլ ծրագիր օգտագործում է այդ port-ը։
`server.js` ֆայլում գտիր `const PORT = 5000;` և փոխիր՝ `const PORT = 5001;`

### Installation failed (npm install error)
→ Ստուգիր ինտերնետ կապը և փորձիր նորից։

## 📁 Ֆայլերի բացատրություն

| Ֆայլ | Ինչ է անում |
|------|-------------|
| `Start-BookingApp.bat` | **Սա՛ է որ սեղմում ես** գործարկելու համար |
| `server.js` | Հիմնական ծրագիրը (backend + frontend միասին) |
| `package.json` | Dependencies-ի ցուցակ |
| `bookings.db` | Տվյալների բազան (ավտոմատ ստեղծվում է) |
| `node_modules/` | Գրադարաններ (ավտոմատ ներբեռնվում են) |

## 💾 Տվյալների պահպանում

Բոլոր ամրագրումները պահվում են **`bookings.db`** ֆայլում՝ նույն թղթապանակում։

- **Backup անելու համար:** Պարզապես պատճենիր `bookings.db` ֆայլը այլ տեղ։
- **Ջնջելու համար (սկսել զրոյից):** Ջնջիր `bookings.db` ֆայլը։ Հաջորդ գործարկման ժամանակ նոր ու դատարկ բազա կստեղծվի։

---

## 🌐 Ինչպես օգտագործել ցանցում (որ ուրիշ համակարգիչներից էլ մուտք գործեն)

Եթե ուզում ես որ նույն Wi-Fi-ին միացած այլ սարքերից (հեռախոս, նոութբուք) մուտք գործեն քո համակարգչում աշխատող ծրագրին՝

1. Գտիր քո համակարգչի IP հասցեն՝
   - Բացիր **CMD** (Command Prompt)
   - Գրիր՝ `ipconfig`
   - Գտիր **IPv4 Address**-ը (օրինակ՝ `192.168.1.42`)

2. Այլ սարքից բրաուզերում մուտքագրիր՝ `http://192.168.1.42:5000`

3. Եթե չի աշխատում, Windows Firewall-ը կարող է արգելափակել։ Թույլատրիր Node.js-ին՝
   - Windows Defender Firewall → Allow an app → Node.js → **Private ✓**

## 🔧 Կարգավորումներ

Ժամանակահատվածները փոխելու համար (օրինակ՝ ավելացնել 09:00 կամ 18:00)՝

`server.js` ֆայլում գտիր այս տողը՝
```javascript
const TIME_SLOTS = ['10:00', '11:00', '12:00', '13:00', '14:00', '15:00', '16:00', '17:00'];
```
և փոխիր ըստ ցանկության։
